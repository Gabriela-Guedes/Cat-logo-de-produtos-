# Catálogo de Roupas e Acessórios

Projeto final da disciplina Desenvolvimento de Aplicações para Internet
Professor: Diogo – 2025

Catálogo interativo de loja virtual desenvolvido 100% com tecnologias front-end puras (sem Bootstrap, React ou qualquer framework).

## O que o projeto faz
- Carrega 30 produtos de um arquivo JSON via AJAX + jQuery
- Busca em tempo real
- Filtro por categoria (botões gerados automaticamente)
- Ordenação por nome e preço
- Modal com detalhes do produto
- Contador de itens exibidos + botão "Limpar filtros"
- Totalmente responsivo (mobile-first)

## Tecnologias usadas
- HTML5
- CSS3 (Grid, Flexbox, animações)
- JavaScript 
- jQuery (obrigatório na disciplina)
- JSON local

## Minha contribuição principal
Criei 100% o arquivo `data/produtos.json` com 30 produtos completos (id, nome, categoria, preço, estoque, descrição e imagens).
